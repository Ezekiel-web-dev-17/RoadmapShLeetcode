## Solutions

### Arrays

#### [Two Sum](Arrays/TwoSum/TwoSum.md)

- **Difficulty:** Easy
- **Description:** Find two numbers in an array that add up to a target value and return their indices.
- **Time Complexity:** O(n)
- **Space Complexity:** O(n)

**Example:**

```javascript
Input: ((nums = [2, 7, 11, 15]), (target = 9));
Output: [0, 1];
```
